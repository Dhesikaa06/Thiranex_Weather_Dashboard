# 🌦 Thiranex Task 4 - Weather Dashboard

## Project Overview

This project was developed as **Task 4** of the **Thiranex Web Development Internship**.

The application fetches real-time weather information from a public REST API using asynchronous JavaScript and displays live weather metrics based on the city entered by the user.

---

## Features

- Search weather by city
- Fetch API
- Async / Await
- REST API Integration
- Dynamic DOM Manipulation
- Error Handling
- Responsive UI
- Temperature
- Humidity
- Wind Speed
- Weather Condition
- Country
- Loading Indicator

---

## Technologies

- HTML5
- CSS3
- JavaScript
- Fetch API
- OpenWeatherMap API
